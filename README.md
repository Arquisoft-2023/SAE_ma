# SAE_ma

# Requerimientos
```
npm 7+
```

# Uso
1. Instalar expo 
```
npm install -g expo-cli
```
2. Instalar las dependencias de nodejs que utiliza el proyecto:
```
npm i
npm install
```
3. Ejecutar el proyecto:
```
Abre tu terminal y navega al directorio de tu proyecto ejecutando el siguiente comando:

cd SAE_ma

Una vez que estés dentro del directorio de tu proyecto, puedes iniciar tu aplicación Expo ejecutando el siguiente comando:

npm start

Esto abrirá una ventana del navegador con la interfaz de Metro Bundler y generará un código QR que puedes escanear con la aplicación Expo Go en tu dispositivo móvil.

Para ejecutar tu aplicación en un dispositivo Android, asegúrate de tener un emulador de Android abierto o un dispositivo Android conectado a tu computadora y ejecuta el siguiente comando:

npm run android

Para ejecutar tu aplicación en un simulador de iOS (requiere macOS), ejecuta el siguiente comando:

npm run ios

Este comando abrirá el simulador de iOS y cargará tu aplicación.

Si deseas ejecutar tu aplicación en la web, simplemente ejecuta el siguiente comando:

npm run web

Esto iniciará tu aplicación en un servidor de desarrollo y podrás acceder a ella a través de tu navegador web.
```
4. Cómo acceder a tu aplicación Expo:
```
Para acceder a tu aplicación Expo que se está ejecutando localmente, sigue los pasos a continuación:

Asegúrate de que tu dispositivo móvil esté conectado a la misma red Wi-Fi que tu computadora.

Instala la aplicación Expo Go en tu dispositivo móvil. Puedes encontrarla en la App Store para dispositivos iOS o en la Google Play Store para dispositivos Android.

Abre la aplicación Expo Go en tu dispositivo móvil.

En la aplicación Expo Go, toca el botón "Scan QR Code" (Escanear código QR).

En tu terminal, verás un código QR generado por Expo. Utiliza la cámara de tu dispositivo móvil para escanear ese código QR.

Una vez escaneado, Expo Go detectará la aplicación y la cargará en tu dispositivo móvil.

¡Listo! Ahora podrás ver y utilizar tu aplicación Expo en tu dispositivo móvil. Recuerda que la aplicación Expo Go y tu computadora deben estar conectadas a la misma red Wi-Fi para que esto funcione correctamente.
```
